【 請求書自動作成システム 】

■ アプリ概要
本アプリは、取引先管理・商品管理・受注管理・請求書発行を一元化する業務効率化システムです。
ExcelやWordで分散していた請求業務を一つのシステムに集約し、転記作業の削減・入力ミス防止・業務の可視化を実現します。

■ URL
https://invoice-cleate.onrender.com/

■ 制作背景
小規模事業者や中小企業は依然としてExcelや手書きによる請求業務・管理を行っている場合が多く、以下の課題が残されています。

・転記作業による工数増大
・入力ミスや請求漏れの発生
・情報の属人化
・請求状況の可視化不足

これらの課題を解決するため、受注から請求までを一元管理できるシステムを開発しました。
また、本アプリはシステムエンジニアへのキャリアチェンジを目的としたポートフォリオとして制作しています。

■ 主な機能

→認証機能
・ユーザー登録機能
・ログイン機能

→登録機能
・顧客登録／編集機能
・商品・サービス登録／編集機能
・売上伝票登録／編集機能

→管理機能
・顧客一覧表示機能
・商品・サービス一覧表示機能
・売上伝票一覧表示機能
・売上履歴一覧表示機能

→出力機能
・請求書作成機能
・PDF出力機能
・印刷機能

■ 使用技術
・フロントエンド：HTML / CSS
・バックエンド：Ruby on Rails
・データベース：PostgreSQL
・インフラ：Render

■ 工夫した点
・HTML・CSSを調整し、「見やすさ・入力しやすさ」を意識したUI設計
・モバイル端末でも利用できるようレスポンシブ対応を実装
・業務フロー（受注 → 請求）に沿った画面遷移設計

■ 苦労した点
・HTMLとCSSの組み合わせによるレイアウト調整に苦労し、試行錯誤を繰り返しながら改善
・Renderへのデプロイ時に、環境差異や設定ミスによるエラー対応に時間を要した

■ 今後の改善点
・UI/UXのさらなる改善
・入金管理機能の追加
・データ分析機能（売上推移の可視化など）の実装
・テストコードの追加による品質向上

■ 対象ユーザー
・Excelや手書き管理に限界を感じている小規模事業者
・業務効率化を図りたい中小企業
・請求業務のデジタル化を検討している方
